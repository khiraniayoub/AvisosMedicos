# 🤖 Guía Rápida - Crear Bot de Telegram

## Paso 1: Abrir Telegram

Tienes dos opciones:
- **Opción A**: Usar Telegram en tu móvil (más fácil)
- **Opción B**: Usar Telegram Web (https://web.telegram.org)

## Paso 2: Buscar BotFather

1. En el buscador de Telegram, escribe: **@BotFather**
2. Haz clic en el contacto verificado (tiene una marca azul ✓)

## Paso 3: Crear el Bot

Envía estos comandos uno por uno:

### 1. Iniciar conversación
```
/start
```

### 2. Crear nuevo bot
```
/newbot
```

### 3. Nombre del bot
Cuando te pregunte el nombre, escribe algo como:
```
Vithas Avisos Bot
```

### 4. Username del bot
Cuando te pregunte el username, escribe algo como:
```
vithas_avisos_bot
```
*(Debe terminar en "bot")*

## Paso 4: Guardar el Token

BotFather te dará un mensaje con el **token**. Se verá así:
```
Done! Congratulations on your new bot...
Use this token to access the HTTP API:
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

**¡COPIA ESE TOKEN!** Lo necesitaremos en el siguiente paso.

---

## ¿Listo con el token?

Una vez tengas el token, dime y te ayudo a configurarlo en la aplicación.
